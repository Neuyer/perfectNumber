## Stack
* Java 21
* Maven
* Spring boot 3.3.5
* Postgres (Banco de dados)


## Instalação

Para instalar o projeto localmente, siga os passos abaixo: <br>

**Certifique-se de ter o Docker instalado em sua máquina**

**LINUX**
```bash
git clone https://github.com/Neuyer/perfectNumber.git
cd perfectnumber
docker-compose up
```
**WINDOWS**
```bash
git clone https://github.com/Neuyer/perfectNumber.git
cd perfectnumber
docker-compose up
```
